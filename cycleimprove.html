Cycle Time Improvement

January 2022

This cell at work performs a simple task but it feeds a main line. Due to some performance issues the cell was running behind and not able to keep up with the main line. I was tasked with improving the cycle time if possible. As you can see in the before video, the robot is taking the long path by starting with the middle part instead of working from the outide in. I retaught the path to improve cycle time and get rid of unwanted movement. This had to be performed for 3 different models and verified.
